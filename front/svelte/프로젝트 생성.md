# Svelte.kit 프로젝트 설정

## Svelte.kit 프로젝트 생성

예전에는 아래와 같이 명령어를 사용하여 프로젝트를 생성했던 경험이 있다. 
```bash
npm create svelte@latest my-app
```

하지만, 지금은 위 명령어가 아닌 다른 방법으로 생성을 해야 한다고 말을 한다.

[프로젝트 생성 관련 공식 문서](https://svelte.dev/docs/kit/creating-a-project)

```bash
npx sv create my-app
```

공식 문서에서 제공하는 정보를 바탕으로 위 명령어를 사용하여 프로젝트 생성을 진행하면 된다.

```bash
┌  Welcome to the Svelte CLI! (v0.9.0)
│
◆  Which template would you like?
│  ● SvelteKit minimal (barebones scaffolding for your new app)
│  ○ SvelteKit demo
│  ○ Svelte library
│
◆  Add type checking with TypeScript?
│  ● Yes, using TypeScript syntax
│  ○ Yes, using JavaScript with JSDoc comments
│  ○ No
│
◆  What would you like to add to your project? (use arrow keys / space bar)
│  ◻ prettier
│  ◻ eslint
│  ◻ vitest
│  ◻ playwright
│  ◻ tailwindcss
│  ◻ sveltekit-adapter
│  ◻ devtools-json
│  ◻ drizzle
│  ◻ lucia
│  ◻ mdsvex
│  ◻ paraglide
│  ◻ storybook (frontend workshop - https://storybook.js.org)
│
◆  tailwindcss: Which plugins would you like to add?
│  ◻ typography (@tailwindcss/typography)
│  ◻ forms
│
◆  sveltekit-adapter: Which SvelteKit adapter would you like to use?
│  ○ auto
│  ○ node
│  ○ static
│  ● vercel (@sveltejs/adapter-vercel)
│  ○ cloudflare
│  ○ netlify
│
◆  Which package manager do you want to install dependencies with?
│  ○ None
│  ● npm
│  ○ yarn
│  ○ pnpm
│  ○ bun
│  ○ deno
└
```

위 처럼 각 단계별 설정 값을 설정해주면서 진행하면 쉽게 설정이 가능해진다.
